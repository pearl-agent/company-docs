## Ideas
- Cross-session context hints: when asking for credentials in one session and receiving them in another, there should be a way to link the request to the delivery
- Brainstorm log automation: daily cron at midnight UTC that reads the day's memory file and generates the brainstorm log automatically
- company-docs onboarding: new agents should be told about company-docs during setup, not discover it later
- Consider expanding brainstorm log data sources beyond memory files — session transcripts, git commits, Slack activity could all feed in

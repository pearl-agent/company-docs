   - https://stytch.com/blog/guide-to-authentication-for-the-openai-apps-sdk/ — Detailed guide on ChatGPT Authorization Code + PKCE flow. User signs in through authorization_endpoint, approves scopes, ChatGPT exchanges code.

   - https://relevanceai.com/agent-templates-tasks/employee-onboarding-ai-agents — Relevance AI on agent onboarding templates. Personalized, scalable solutions for integrating new agent "hires."

3. **Query:** "Slack webhook mode vs socket mode provisioning signingSecret"

   - https://hookdeck.com/webhooks/platforms/guide-to-slack-webhooks-features-and-best-practices — Socket Mode avoids timeout constraints but webhook mode is more standard for production deployments with proper verification.
